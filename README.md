# 🔬 Research Agent
Research Agent is a web application that allows users to search research papers from arXiv and automatically generates AI-powered summaries for each paper. It combines a React frontend, Node.js backend, and Google Gemini AI for summarization.


## Features
- Search research papers by topic from arXiv
- AI-generated summaries in 3 concise bullet points
- Clean, user-friendly React frontend
- Node.js + Express backend for API handling
- Fully asynchronous, handles multiple papers

## Tech Stack
- **Frontend:** React, HTML, CSS, JavaScript
- **Backend:** Node.js, Express, Axios
- **AI Integration:** Google Generative AI (Gemini API)
- **Data Parsing:** xml2js (XML → JSON)
- **Other:** CORS for frontend-backend communication
